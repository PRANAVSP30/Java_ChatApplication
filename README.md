### Project Overview

This project implements a real-time chat application using:

  Java Sockets

  Multi-threading

  Client–Server Architecture

Each connected user communicates through the central server.

Every client runs in a separate thread.

### Features 

  Username system
  Join/Leave broadcast messages
  Timestamp for all messages
  Handles unlimited clients

### System Architecture

  Server

    Accepts new clients

    Spawns a thread per client

    Broadcasts messages to all users

  Clients

    Connect to server

    Send & receive messages

    Run read/write threads simultaneously

### Tech Stack
- Java (Core Java)
- Socket Programming (TCP)
- Multi-threading for concurrent clients
- Real-time client–server communication
- Tools: VS Code, Java JDK 25, Git & GitHub
